# คู่มือการใช้งาน Admin — F-Secret AI Art Gallery

## เข้าสู่หลังบ้าน

1. เปิด URL เว็บไซต์
2. เติม `/admin` ต่อท้าย URL เช่น `https://FsecretAi.vercel.app/admin`
3. ใส่รหัสผ่านที่ตั้งไว้
4. กด "เข้าสู่ระบบ"

## สร้าง Catalog ใหม่

1. ไปที่เมนู **Catalogs**
2. กรอกชื่อ Catalog ในช่อง "ชื่อ Catalog"
3. (ไม่บังคับ) ใส่คำอธิบายและลำดับ
4. กด "สร้าง"
5. Catalog ใหม่จะแสดงในรายการ สามารถแก้ไขชื่อหรือลำดับได้ภายหลัง

## อัปโหลดรูป

1. ไปที่เมนู **Upload**
2. เลือก Catalog ปลายทางจาก dropdown
3. ลากรูปมาวางในกรอบ หรือกดเลือกไฟล์จากเครื่อง
   - เลือกได้หลายรูปพร้อมกัน
4. รอ preview โหลด
5. กด "อัปโหลด"
6. ระบบจะแสดง Progress Bar จนกว่าจะอัปเสร็จ
7. อัปเสร็จแล้วจะ redirect ไปหน้า **Images** อัตโนมัติ

## จัดการรูป

### ย้ายรูปไป Catalog อื่น

1. ไปที่เมนู **Images**
2. เลือกรูปที่ต้องการย้าย (คลิก checkbox ที่มุมรูป)
3. เลือก Catalog เป้าหมายจาก dropdown ด้านบน
4. กด "ย้ายรูปที่เลือก"

### ลบรูป

1. ไปที่เมนู **Images**
2. เลือกรูปที่ต้องการลบ
3. กด "ลบรูปที่เลือก"
4. ยืนยันการลบ

### เรียงลำดับรูป

1. ไปที่เมนู **Images**
2. ใช้ filter เลือก Catalog ที่ต้องการ
3. แก้ไขตัวเลขลำดับที่ใต้แต่ละรูป
4. กด "บันทึกลำดับ" (ถ้ามี)

หรือเรียงลำดับได้ที่เมนู **Catalogs** → คลิกชื่อ Catalog → จัดการลำดับรูป

## ตั้งค่าเว็บ

1. ไปที่เมนู **Settings**
2. แก้ไขข้อมูลตามต้องการ:
   - **ชื่อเว็บ** — แสดงที่ tab browser และหน้าแรก
   - **คำอธิบาย** — แสดงที่หน้าแรก
   - **Line ID** — แสดงที่หน้าติดต่อ
   - **Line QR Code URL** — ลิงก์รูป QR Code (อัปโหลดที่ Cloudinary แล้วเอา URL มาใส่)
   - **Logo URL** — ลิงก์รูปโลโก้ (อัปโหลดที่ Cloudinary แล้วเอา URL มาใส่)
3. กด "บันทึกการตั้งค่า"

## อัปโหลดรูปเพิ่มทีหลัง

ทำซ้ำตามขั้นตอน **อัปโหลดรูป** ข้างต้นได้ตลอด ไม่จำกัดจำนวนครั้ง

## ออกจากระบบ

1. คลิกเมนู **Logout** ที่ Sidebar
2. หรือปิด browser tab

## ข้อควรระวัง

- รหัสผ่าน Admin เก็บใน Environment Variables ของ Vercel หากลืมรหัสผ่านต้องแก้ไขใน Vercel Dashboard
- การลบ Catalog จะลบรูปทั้งหมดใน Catalog นั้นด้วย ควรย้ายรูปออกก่อนลบ
- รูปที่อัปโหลดจะเก็บที่ Cloudinary หากลบรูปในหน้า Images ระบบจะลบจาก Cloudinary ด้วย
